AgentUI SVG components

Five SVG-native Web Components rendering one realistic agent run. Click nodes, items, and states — selections cross-highlight between components, and every event is logged below. Edit fixtures.js to change the data.

Agent Workflow Canvas <agentui-workflow-canvas>

drag to pan · wheel to zoom
Usage 
const el = document.createElement("agentui-workflow-canvas");
el.data = { nodes: [...], edges: [...] };
el.addEventListener("agentui:select", (e) => e.detail); // { id, kind, data }

Tool Call Timeline <agentui-tool-timeline>

Usage 
el.setAttribute("density", "compact"); // or "expanded"
el.data = { items: [{ id, label, status, durationMs, detail }] };

Review Checkpoint <agentui-review-checkpoint>

Usage 
el.data = { title, description, level: "high", summary };
el.addEventListener("agentui:decision", (e) => e.detail); // { action, note? }

Agent Memory Map <agentui-memory-map>

Usage 
el.setAttribute("layout", "grouped"); // or "default"
el.data = { nodes: [{ id, label, type }], links: [{ from, to, strength }] };
el.addEventListener("agentui:remove", (e) => e.detail.id);

State Machine Viewer <agentui-state-machine>

Usage 
el.setAttribute("layout", "radial"); // or "horizontal"
el.data = { states: [...], transitions: [...], active: "running" };
el.setActive("waiting");
